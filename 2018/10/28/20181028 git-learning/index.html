git 基本操作

初始化
git init
提交暂存区
git add readme.txt
提交仓库
git commit -m "wrote a readme file"
查看状态
git status
查看修改的内容
git diff readme.txt
删除文件

版本控制

版本历史记录
git log
版本回退
git reset --hard HEAD^ 或 git reset --hard version_ID
版本回退历史
git reflog
撤销操作(到上次git add或git commit)
git checkout -- readme.txt

git 远程库操作

本地库推送远程

  1. 在github创建库 learngit
  2. 本地关联
    git remote add origin git@github.com:baihaobo/learngit.git
  3. 本地库推送到github
    第一次
    git push -u origin master
    以后
    git push origin master

远程库推送本地

git clone git@github.com:michaelliao/gitskills.git
